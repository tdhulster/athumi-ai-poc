# travel-bot

## 1 - Install python and dependencies

### Install venv
`python3 -m venv .venv`

### Activate venv
`source .venv/bin/activate`

### Deactivate 
`deactivate`

### Install requirements
`pip install -r requirements.txt`

### Save dependencies in requirements file 
`pip freeze > requirements.txt`
